For the PDFs, see the online Mathomatic User Guide and Command Reference PDF file in a file called "manual.pdf", or all Unix/Linux man pages for Mathomatic in one PDF file called "bookman.pdf". PDFs are generally used when you want to print out a hardcopy 8.5-inch by 11-inch manual, but they can be browsed with a PDF document viewer and saved to disk easily, too.
